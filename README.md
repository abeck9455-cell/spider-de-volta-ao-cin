# Arduino Spider Robot 🕷️

A simple four-legged spider robot built with Arduino.

FEATURES

- Forward movement
- Servo-based walking gait
- Easy to customize

HARDWARE/ELETRONICS

- Arduino Nano
- arduino Nano Shield
- 12 Servo motors 
- external voltage of 7-8V
- 3D printed spider robot 



SOFTWARE

- Arduino IDE
- Required libraries:
  - Servo.h



CIRCUIT

- 2,3,4,5 consections represent the pivot servos
- 6,7,8,9 connections represent the lift servos
- 10,11,12,13 connections represent the paws
